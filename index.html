Web コンポーネント入門

Webコンポーネントは、再利用可能なカスタム要素を作成するためのWeb標準技術です。

基本概念

1. カスタム要素

HTMLに独自のタグを定義できます。

2. Shadow DOM

スタイルとマークアップをカプセル化し、他の要素と分離します。

3. リアクティブプロパティ

データの変更を自動的にUIに反映します。

シンプルな例

カウンターコンポーネントで基本を学びましょう。

import { LitElement, html, css } from 'lit';

class CounterButton extends LitElement {
  static properties = {
    count: { type: Number }
  };

  static styles = css\`
    button {
      padding: 10px 20px;
      font-size: 16px;
      cursor: pointer;
    }
  \`;

  constructor() {
    super();
    this.count = 0;
  }

  render() {
    return html\`
      <button @click=\${() => this.count++}>
        クリック数: \${this.count}
      </button>
    \`;
  }
}

customElements.define('counter-button', CounterButton);

ボタンをクリックすると、カウントが増えます。プロパティの変更が自動的にUIに反映されます。

使い方

定義したコンポーネントは、通常のHTML要素のように使用できます。

<counter-button></counter-button>

主な特徴

  • フレームワーク不要で動作
  • どのプロジェクトでも再利用可能
  • 標準のWeb技術に基づく
